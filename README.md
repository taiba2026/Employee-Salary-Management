# Employee Salary Management System

## Description
This is a simple Java Swing project to manage employee records.

## Features
- Add Employee
- View Employee
- Delete Employee
- Calculate Net Salary
- Show Highest Paid Employee
- File Handling (employees.txt)

## Technologies Used
- Java (Swing)
- MySQL
- JDBC

## How to Run
1. Open project in IntelliJ IDEA
2. Run Login.java
3. Use username: admin, password: 1234
